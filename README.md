# Python Foundations

A collection of Python utility scripts built as part of the Applied AI Engineer bootcamp.

## Projects

### CSV Analyser
Reads a CSV file and outputs summary statistics including row count, column names, and basic analysis.

### Word Counter
Accepts a text input and returns word frequency counts.

## Setup

```bash
git clone https://github.com/KoketjoM/python-foundations.git
cd python-foundations
C:\Python312\python.exe -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

## Running Tests

```bash
pytest
```

## Tech Stack
- Python 3.12
- pytest
